# Vraj Patel — Portfolio

A modern, full-stack developer portfolio built with **React + Vite**, featuring a Three.js particle background, GSAP scroll animations, and a Framer Motion custom cursor.

🔗 **Live:** [vrajpatel.dev](https://vraj-patel.me)

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Framework | React 19 + Vite 7 |
| Styling | Tailwind CSS v4 (compiled) |
| 3D / Particles | Three.js |
| Scroll Animations | GSAP + ScrollTrigger |
| Entrance Animations | Framer Motion |
| Icons | Lucide React |
| Deployment | Vercel |

## Projects Featured

- **CineMaster** — MERN movie discovery platform with JWT auth and TMDB integration
- **WatchWise** — Full-stack movie/TV app with AI chatbot recommendations (OpenRouter)
- **MarkME** — AI face-recognition attendance system (FastAPI + InsightFace + MERN)
